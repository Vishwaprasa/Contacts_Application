# 📒 iFRIEND Contact Organizer

Welcome to the **iFRIEND Contact Organizer**, a Java-based contact management system that allows users to store, update, delete, search, and sort contact details such as name, phone number, company, salary, and birthday.

---

## 💡 Features

- 🔢 Auto-generate unique contact IDs
- 📥 Add contacts with validation (phone number, salary, birthday)
- 🔄 Update contact details individually
- 🔍 Search contacts by name or phone number
- 🗑️ Delete contacts
- 📃 List contacts sorted by:
  - Name (A–Z)
  - Salary (Ascending)
  - Birthday (Chronologically)
- 🧹 Clears console output for cleaner interaction (Windows & Linux supported)

---

## 🛠️ Technologies Used

- **Java** (Standard Edition)
- **Local Arrays** for data storage (no external libraries or DB)
- **Console-based UI**
